PHP предоставляет несколько встроенных структур данных и инструментов для работы с ними. Вот основные из них:

1. Массивы (Arrays)

Гибкая структура, которая может работать как:

  • Индексированный массив (список):

    php
    Copy
    $fruits = ['Яблоко', 'Банан', 'Апельсин'];
echo $fruits[0]; // Яблоко

  • Ассоциативный массив (словарь/хэш-таблица):

    php
    Copy
    $user = [
    'name' => 'Иван',
    'age' => 30,
    'city' => 'Москва'
];
echo $user['city']; // Москва

  • Многомерный массив:

    php
    Copy
    $users = [
    ['id' => 1, 'name' => 'Анна'],
    ['id' => 2, 'name' => 'Петр']
];

Основные операции:

php
Copy
array_push($fruits, 'Груша'); // Добавить в конец
$last = array_pop($fruits);     // Удалить последний
array_unshift($fruits, 'Манго'); // Добавить в начало

2. SPL (Standard PHP Library)

Встроенная библиотека для работы с продвинутыми структурами данных:

  • Стек (LIFO) через SplStack:

    php
    Copy
    $stack = new SplStack();
$stack->push('Элемент 1');
$stack->push('Элемент 2');
echo $stack->pop(); // Элемент 2

  • Очередь (FIFO) через SplQueue:

    php
    Copy
    $queue = new SplQueue();
$queue->enqueue('Задача 1');
$queue->enqueue('Задача 2');
echo $queue->dequeue(); // Задача 1

  • Кучи (Heaps) через SplHeap:

    php
    Copy
    $heap = new SplMaxHeap(); // Максимальная куча
$heap->insert(30);
$heap->insert(10);
echo $heap->extract(); // 30

3. Объекты и Классы

Создание собственных структур данных через ООП:

php
Copy
class Node {
    public $value;
    public $next;
    public function __construct($value) {
        $this->value = $value;
        $this->next = null;
    }
}

// Пример связного списка
class LinkedList {
    private $head;
    public function add($value) {
        $node = new Node($value);
        if ($this->head === null) {
            $this->head = $node;
        } else {
            $current = $this->head;
            while ($current->next !== null) {
                $current = $current->next;
            }
            $current->next = $node;
        }
    }
}

4. Множества (Sets)

PHP не имеет встроенного Set, но его можно эмулировать через ассоциативные массивы:

php
Copy
$set = [];
$set['элемент1'] = true;
$set['элемент2'] = true;
if (isset($set['элемент1'])) {
    echo 'Элемент1 есть!';
}

5. Строки (Strings)

Строки в PHP могут обрабатываться как массивы символов:

php
Copy
$text = "Hello";
echo $text[1]; // e

6. Структуры данных PHP 7.4+

С версии 7.4 доступны типизированные свойства:

php
Copy
class TreeNode {
    public int $value;
    public ?TreeNode $left = null;
    public ?TreeNode $right = null;

    public function __construct(int $value) {
        $this->value = $value;
    }
}

Рекомендации:

  • Используйте массивы для простых коллекций.

  • Для сложных операций (очереди, стеки) применяйте SPL.

  • Для полного контроля создавайте классы (например, деревья, графы).

  • Учитывайте производительность: вставка в начало массива (array_unshift) медленнее, чем в конец (array_push).

Примеры задач и структур:

Работа с приоритетами SplPriorityQueue
