Работа с файлами и директориями

  1. ls — список файлов и директорий.

    • ls -l — подробный список.

    • ls -a — показать скрытые файлы.

  2. cd — сменить директорию.

    • cd ~ — перейти в домашнюю директорию.

    • cd .. — перейти на уровень выше.

  3. pwd — показать текущую директорию.

  4. mkdir — создать директорию.

    • mkdir -p dir1/dir2 — создать вложенные директории.

  5. rm — удалить файл или директорию.

    • rm file.txt — удалить файл.

    • rm -r dir — удалить директорию рекурсивно.

  6. cp — копировать файл или директорию.

    • cp file1 file2 — копировать файл.

    • cp -r dir1 dir2 — копировать директорию.

  7. mv — переместить или переименовать файл/директорию.

    • mv file1 file2 — переименовать файл.

    • mv file1 dir/ — переместить файл.

  8. touch — создать пустой файл или обновить время его изменения.

  9. cat — вывести содержимое файла.

  10. less или more — просмотр файла постранично.

  11. head — показать начало файла.

    • head -n 10 file.txt — первые 10 строк.

  12. tail — показать конец файла.

    • tail -f file.log — следить за изменениями файла в реальном времени.

  13. find — поиск файлов.

    • find /path -name "*.txt" — найти все .txt файлы.

  14. grep — поиск текста в файлах.

    • grep "pattern" file.txt — найти строки, содержащие "pattern".

    • grep -r "pattern" /path — рекурсивный поиск в директории.

Работа с процессами

  1. ps — список процессов.

    • ps aux — подробный список всех процессов.

  2. top или htop — мониторинг процессов в реальном времени.

  3. kill — завершить процесс.

    • kill PID — завершить процесс по ID.

    • kill -9 PID — принудительное завершение.

  4. pkill — завершить процесс по имени.

    • pkill process_name.

  5. bg и fg — управление фоновыми и активными задачами.

Сеть

  1. ping — проверить доступность хоста.

    • ping google.com.

  2. curl или wget — загрузка файлов из интернета.

    • curl -O URL.

    • wget URL.

  3. ssh — подключение к удаленному серверу.

    • ssh user@host.

  4. scp — копирование файлов через SSH.

    • scp file.txt user@host:/path.

  5. netstat или ss — информация о сетевых соединениях.

    • netstat -tuln.

Управление пакетами

  1. apt (Debian/Ubuntu):

    • apt update — обновить список пакетов.

    • apt install package — установить пакет.

    • apt remove package — удалить пакет.

  2. yum или dnf (CentOS/RHEL/Fedora):

    • yum install package.

    • yum remove package.

  3. pip (Python):

    • pip install package.

    • pip uninstall package.

Системная информация

  1. uname — информация о системе.

    • uname -a — вся информация.

  2. df — информация о дисковом пространстве.

    • df -h — в удобном формате.

  3. du — размер файлов и директорий.

    • du -sh dir — суммарный размер директории.

  4. free — информация о памяти.

    • free -h — в удобном формате.

Разработка

  1. git — система контроля версий.

    • git clone URL — клонировать репозиторий.

    • git status — статус репозитория.

    • git add file — добавить файл в индекс.

    • git commit -m "message" — зафиксировать изменения.

    • git push — отправить изменения на сервер.

    • git pull — получить изменения с сервера.

  2. make — сборка проектов.

    • make — запуск сборки.

    • make clean — очистка.

  3. gcc или g++ — компиляция C/C++.

    • gcc file.c -o output.

  4. python или python3 — запуск Python скриптов.

    • python script.py.

Права доступа

  1. chmod — изменить права доступа.

    • chmod 755 file — установить права.

  2. chown — изменить владельца файла.

    • chown user:group file.

Архивация

  1. tar — работа с архивами.

    • tar -cvf archive.tar dir — создать архив.

    • tar -xvf archive.tar — распаковать архив.

  2. zip и unzip — работа с ZIP архивами.

    • zip archive.zip file — создать ZIP.

    • unzip archive.zip — распаковать ZIP.

Переменные окружения

  1. export — установить переменную окружения.

    • export VAR=value.

  2. echo — вывести значение переменной.

    • echo $VAR.

Другое

  1. man — справочная информация о команде.

    • man ls.

  2. alias — создать псевдоним для команды.

    • alias ll='ls -la'.

  3. history — история команд.

    • history | grep "command".
